# 🧠 Projeto de Machine Learning - Previsão de Preço de Casas

Este projeto aplica técnicas de Ciência de Dados e Machine Learning para prever o valor de casas com base na renda média de uma região.

---

## 📊 Objetivo

Construir um modelo preditivo capaz de identificar padrões entre renda média e preço das casas, apoiando análises orientadas a dados.

---

## 🗂️ Dataset

- Fonte: dataset público de habitação  
- Variáveis utilizadas:
  - `median_income` → renda média  
  - `median_house_value` → valor das casas  

---

## ⚙️ Tecnologias Utilizadas

- Python  
- Pandas  
- Matplotlib  
- Scikit-learn  

---

## 🔎 Etapas do Projeto

- Coleta de dados  
- Limpeza e tratamento  
- Análise exploratória  
- Visualização de dados  
- Treinamento de modelo (Regressão Linear)  
- Avaliação de desempenho  
- Geração de previsões  

---

## 📈 Visualização

O projeto inclui análise gráfica da relação entre renda média e valor das casas, permitindo identificar tendências relevantes.

---

## 🤖 Modelo de Machine Learning

- Algoritmo: Regressão Linear  
- Divisão dos dados:
  - Treino: 80%  
  - Teste: 20%  

---

## 📊 Resultados

O modelo foi capaz de capturar a relação entre renda média e valor das casas, apresentando desempenho consistente.
